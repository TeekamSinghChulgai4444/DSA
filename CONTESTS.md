## 💡 Tips for Contest Prep

1. Read all problems before starting — pick the easiest first
2. Don't spend more than 20 minutes stuck on one problem
3. Test edge cases before submitting
4. After contest: upsolve all problems you couldn't solve
5. Read editorial for any problem you found hard
